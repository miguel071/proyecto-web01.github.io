JAVA

Java: Es un lenguaje de programación que se usa para crear aplicaciones. Puedes usar Java para hacer aplicaciones para Android, programas en servidores y más.

C++

Es un lenguaje de programación que se usa para hacer software que necesita ser muy rápido y eficiente, como videojuegos, sistemas operativos y aplicaciones de alto rendimiento.

CSS

CSS (Cascading Style Sheets): Es el lenguaje que usas para darle estilo a tu página web. Con CSS, puedes cambiar cómo se ven los elementos creados con HTML, como colores, fuentes y disposición en la página.

HTML

HTML (HyperText Markup Language): Es el lenguaje que usas para construir la estructura de una página web. Con HTML, decides qué elementos van en tu página, como títulos, párrafos y enlaces.
